# SmashQL
Super Smash Bros. Ultimate display using GraphQL!

## Getting Started

### Installing

Install 

```
npm install
```

Populate PostgreSQL database, build the bundle, and start the server

```
npm run seed
npm run build
npm run start
```

App will run on port 1025


## Built With

* GraphQL
* Apollo
* SCSS
